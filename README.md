# Galería de Retratos a Lápiz 🎨✏️

Sitio web simple y liviano para mostrar retratos hechos a lápiz. Fue creado para una amiga artista como una galería online para exhibir su trabajo.

![Captura del sitio](/screenshot.jpg)

## 🛠 Tecnologías usadas

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [TailwindCSS](https://tailwindcss.com/)
- [SWR](https://swr.vercel.app/) – para una mejor experiencia de carga de datos
- [Supabase](https://supabase.io/) – backend y almacenamiento de imágenes
